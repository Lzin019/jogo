# 🃏 PokéDS - Pokémon TCG Simulator

> Simulador digital do jogo de cartas Pokémon TCG, totalmente desenvolvido para a disciplina de DS. O projeto aplica conceitos de Programação Orientada a Objetos e estruturas de dados, como as pilhas e listas, para gerenciar turnos, batalhas e decks. O foco foi transformar todas as regras em algoritmos, criando um sistema digital funcional.

---

## 🎮 Sobre o Jogo
Este projeto foi criado com o objetivo de traduzir as regras complexas de mesa do Pokémon TCG para o ambiente digital. O jogo gerencia toda a lógica de uma partida, desde o momento de embaralhar o deck até o cálculo de danos e a checagem das condições de vitória.

## ✨ Funcionalidades
* **Gerenciamento de Decks:** Compra de cartas, controle de deck e pilha de descarte.
* **Sistema de Turnos:** Alternância correta entre as fases de preparação, ataque e passagem de turno.
* **Mecânicas de Batalha:** Cálculo de dano dinâmico baseado nos ataques e gerenciamento de HP dos Pokémon.
* **Tipagem de Cartas:** Implementação de cartas do tipo Pokémon, Energia e Treinador com comportamentos distintos.

## 🛠️ Tecnologias e Estruturas de Dados
Para garantir a organização e eficiência do código, foram utilizados os seguintes conceitos:
* **Pilhas (Stacks):** Utilizadas para estruturar o Deck (onde a última carta inserida é a primeira a ser comprada) e a Pilha de Descarte.
* **Listas (Lists):** Utilizadas para gerenciar dinamicamente a Mão do jogador e o Banco de Pokémon ativos/reserva.
* **POO (Programação Orientada a Objetos):** Uso de herança e polimorfismo para a criação das classes de cartas.
